Project Name: ButlerTask

Objective:
Build a productivity web app where users:
- Register/Login
- Add tasks with date & time
- View tasks
- Mark complete
- Detect missed tasks
- Highlight tasks near start time

Tech Stack:
- Next.js (App Router)
- Supabase (Auth + Database)
- Tailwind CSS
- Vercel (Deployment)

Phase 1 Scope:
- Authentication
- Task CRUD
- Highlight within 15 mins
- Auto missed detection
- Dashboard UI
- IST timezone support

Future Phases:
- Email reminders
- AI Butler
- Calendar view
- Analytics
